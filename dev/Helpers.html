Helpers

The Helpers sub-package contains both user interface function and functions useful for many of the internals such as data structure objects.

FaADE.Helpers.NodeType ‚Äî Type
NodeType
  • Used by FirstDerivative() and SecondDerivative() to indicate which node type should be used.
  • Left and Right are used by the SAT() function to determine which side of the SAT to call.

Node types are:

  • Left
  • Internal
  • Right
  • Up
  • Down
